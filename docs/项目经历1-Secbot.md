# 项目经历1

## Secbot-安全测试机器人 · 2024.12-2025.11

**技术选型:** Python, LangChain, SQLite, Typer

---

### 项目描述

面向授权渗透测试的 AI 驱动安全测试平台。采用多智能体架构，基于 LangChain 实现侦察、漏洞扫描与利用、主动防御、Web 研究等模块，支持完整攻击链编排与 Payload 生成；提供 CLI 交互、语音与远程控制，使用 SQLite 持久化对话与配置。

---

### 责任描述

1. 基于 LangChain 实现多智能体框架，设计 ReAct、Plan-Execute、工具调用与记忆增强等模式，支撑渗透测试与 Web 研究场景。

2. 设计并实现渗透测试工具链：信息收集、端口/漏洞扫描、利用引擎与后渗透，支持攻击链编排与 Payload 自动生成。

3. 实现主动防御：网络发现、入侵检测、授权管理、远程执行与文件传输，并生成安全报告。

4. 使用 Typer 构建 CLI，集成语音、爬虫与系统控制工具，基于 SQLite 持久化对话与配置。

5. 编写项目与部署文档、安全警告说明，配置 Docker 部署，保障多环境可部署与可维护性。
